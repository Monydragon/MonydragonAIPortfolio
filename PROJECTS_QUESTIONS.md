## RPG Maker Plugins

## Random Loot System (RPG Maker MV/MZ)

**Current Description:** Does your game have WAY too much Loot to manage like *Titan Quest* or *Disgaea*? Tired of manually tweaking Rewards given by 1000+ Chest Events? Want to stop players from grinding starter mobs by dynamically adjusting drop rates based on what's in their Inventory (and other Variables)? Look no further as **Random Loot System**'s got you covered!

**Random Loot System** (RLS) is a plugin for RPG Maker MV & MZ that allows you to define Loot Tables for Items, Weapons, Armor, Gold and Experience that can be rolled for via Plugin Commands and Notetags. Easily customize the rates, amount, conditions of each Entry in a Loot Table and have those changes reflected across the entire project at a press of a button.

**Questions:**

- What technologies/frameworks were used? (e.g., C#, Unity version, any specific Unity packages?)

  A: RPG Maker MV/MZ, JavaScript

- Is there a GitHub repository? (URL if available)

- A: Not at this time

- Is it available on itch.io
  A: Yes: https://mythatelier.itch.io/random-loot-system

- Any live demo or documentation site?

- What are the relevant features?

- A: 

- **Plugin Features**

  - Create Loot Tables to define sets of possible Rewards. 
  - Configure the odds and conditions of each Item in the Table.
  - Choose Fixed, Percentage or Weighted based loot draw per Table.
  - Replicate Loot rolls exactly for a playthrough by setting a Random Seed.
  - Adds Loot Tables to Enemy and Troop's Battle rewards through Notetags.
  - Plugin Commands that let you roll from a Loot Table on Event Pages.
  - Give Skill/Item Notetags to allow Loot Table rolls on their use.
  - Toggleable Debug Message system to show roll calculations.
